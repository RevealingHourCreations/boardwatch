# Arduino Uno
<img src="ArduinoUno_R3_Front.jpg" height="200"  > 

#### Company : Arduino
#### Microcontroller : AtMega328
#### OS supported : None
#### Memory : 32KB
#### Clock Speed : 16 Mhz
#### Power supply
* Input Voltage : 7-12V
* Connector
  * 2.1mm center-positive plug
  * USB B Female

|Type|Number|Count|

|GPIO|
|ADC|
|PWM|
|SPI|
|I2C|
|WiFi|
|Ethernet|
|HDMI|
