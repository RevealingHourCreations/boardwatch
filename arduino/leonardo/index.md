# Arduino Leonardo
<img src="ArduinoLeonardo.jpg" height="200"  > 

#### Company : Arduino
#### Microcontroller : ATmega32u4 
#### OS supported : None
#### Memory : 32KB
#### Clock Speed : 16 Mhz
#### Power supply
* Input Voltage : 7-12V
* Connector
  * 2.1mm center-positive plug
  * micro-USB
