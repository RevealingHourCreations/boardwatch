# Arduino Duemilanove
<img src="ArduinoUno_R3_Front.jpg" height="200"  > 

#### Company : Arduino
#### Microcontroller : ATmega168, AtMega328
#### OS supported : None
#### Memory : 16kB, 32KB
#### Clock Speed : 16 Mhz
#### Power supply
* Input Voltage : 7-12V
* Connector
  * 2.1mm center-positive plug
  * USB B Female
	
#### Interfaces

|Type|Number|Bus Count|
|---|---|---|
|GPIO|0-13|14|
|ADC|
|PWM|3,5,6,9,10,11|6|
|SPI|10(SS),11(MOSI),12(MISO),13(SCK)|1|
|I2C|A4(SDA),A5(SCL)|1|
|WiFi|
|Ethernet|none| |
|HDMI|None| |
